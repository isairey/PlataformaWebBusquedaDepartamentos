# 🌌 Acerca del proyecto

**Apartment Rental Management System** es una plataforma web creada para optimizar el proceso de búsqueda y administración de apartamentos en renta.

La plataforma permite:

- 🏢 Publicar apartamentos disponibles
- 📍 Buscar propiedades por ubicación
- 💰 Consultar precios de renta
- 👥 Gestionar propietarios e inquilinos
- 📊 Administrar información inmobiliaria
- 💳 Realizar pagos online
- 🔎 Filtrar apartamentos según preferencias
- 🌐 Acceder desde cualquier dispositivo

El sistema fue diseñado pensando en:

- ⚡ Automatización inmobiliaria
- 📱 Acceso rápido y moderno
- 🧠 Gestión eficiente de datos
- 🌍 Experiencia intuitiva
- 🔒 Administración segura

# 🚀 Configuración del proyecto

## 1️⃣ Clonar repositorio

```bash
git clone https://github.com/isairey/PlataformaWebBusquedaDepartamentos.git
```

---

## 2️⃣ Entrar al proyecto

```bash
cd PlataformaWebBusquedaDepartamentos
```

---

## 3️⃣ Instalar dependencias

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Configurar base de datos

Copiar y ejecutar los comandos SQL del directorio:

```bash
database/
```

usando:

```bash
MYSQL Workbench
```

---

## 5️⃣ Iniciar XAMPP

Activar:

- Apache Webserver
- MySQL Server

---

## 6️⃣ Ejecutar proyecto

Abrir:

```bash
main.py
```

Ejecutar en VS Code:

```bash
python main.py
```

---

## 🌐 Abrir aplicación

Aplicación disponible en:

```bash
http://127.0.0.1:5000
```
